# CNN
One simple project uses CNN to classify cifar10 dataset

## Code Environment

### Python >= 3.6
### PyTorch >= 1.10.2
### CUDA >= 12.4 (depends on your GPU)
### Dataset >= CIFAR-10
### PyCharm IDE is recommended
